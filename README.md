PRESCRIPTO – Doctor Appointment Booking Platform A modern MERN Stack web application designed to simplify and digitize doctor appointment management. Prescripto allows patients to easily find, compare, and book appointments with doctors based on specialization, fees, and availability. It also provides dedicated dashboards for Patients, Doctors, and Admins for seamless management.

Features

 User Authentication (JWT-Based) — Secure login/signup for patients, doctors, and admin.
 Appointment Booking — Users can book appointments with their preferred doctors.
 Doctor Comparison — Filter and compare doctors by specialization, consultation fees, and available slots.
 Dedicated Dashboards — Doctor Dashboard, User Dashboard and Admin Dashboard.
Tech Stack

Frontend:

   HTML, CSS, JavaScript
   React.js

Backend:

  Node.js
  Express.js
  MongoDB + Mongoose
  JWT Authentication
  Multer (for future profile uploads)

Other Tools:

  Git & GitHub
  Render (for deployment)
How It Works

1️ User Signup/Login — Authenticate securely using JWT.
2️ Browse Doctors — View doctors by specialization, experience, or fees.
3️ Compare & Book — Choose your preferred doctor and book a slot.
4️ Dashboard Access —
      Patients: Manage upcoming and past appointments.
      Doctors: Manage appointments and update availability.
      Admin: Monitor users, doctors, and appointments.
5️ (Coming Soon) Payment integration and virtual consultation modules.
